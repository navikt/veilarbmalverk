VeilarbMalverk
================

Enkel microservice som holder styr på maler som brukes innad i FO. 

# Komme i gang

Hvordan bygge, teste og kjøre koden viss aktuelt.
```
# bygge
mvn clean install 

# test
mvn test

# starte
# I ditt favoritt IDEA kjør main-metoden i StartLocal.java
```
